# Ciberseguridad — Exploración Autodidacta

> ⚠️ **Incompleta** — faltan áreas y detalles por agregar.

> Período de exploración intensa: ~2019–2023 (12–16 años)
> Naturaleza: completamente autodidacta, sin guía formal

## Áreas exploradas

### Ingeniería inversa
- Modificación de memoria en aplicaciones y juegos (herramientas especializadas)
- Análisis de ejecutables de juegos indie (Undertale, tycoons, etc.)
- Generación de versiones modificadas de software (cracks básicos)

### Scripting y automatización
- Automatización de peticiones HTTP
- Manejo de proxies para evitar limitación de tasa (rate limiting)
- Interacción con APIs web

### OSINT y búsqueda avanzada
- Google Dorks para búsquedas específicas indexadas
- Técnicas de recolección de información pública

### Redes y protocolos
- Análisis básico de tráfico de red
- Conceptos de interceptación de comunicaciones cliente-servidor
- Fundamentos aplicados en el contexto de Roblox (sistema de Remote Events)

### Creación de malware
- Exploración teórica y práctica básica
- Sin llegada a deployment real o daño a terceros conocido

## Contexto y reflexión
Esta fase fue parte de una curiosidad técnica intensa, sin orientación formal ni límites claros. El conocimiento adquirido (automatización, análisis de protocolos, ingeniería inversa) se transfirió posteriormente a contextos legítimos: robótica, simulación biológica, investigación.

| Automatización / scripting | Python para análisis de datos y robótica |
| Análisis de sistemas | Modelado de redes metabólicas |
| Ingeniería inversa | Comprensión de código de modelos SBML |
